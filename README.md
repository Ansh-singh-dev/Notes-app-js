Notes App

A simple Notes App built using HTML, CSS, and JavaScript.
Users can create and delete notes, and the notes are stored in the browser using localStorage, so they remain even after refreshing the page.

🚀 Live Demo

https://ansh-singh-dev.github.io/Notes-app-js/

✨ Features

- Add new notes
- Delete notes
- Notes persist after page refresh using localStorage
- Clean and responsive user interface

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

📂 Project Structure

notes-app-js
├── index.html
├── style.css
├── script.js
└── README.md

⚙️ How It Works

1. The user types a note in the textarea.
2. Clicking Add Note creates a new note card.
3. Notes can be removed using the ✕ delete button.
4. Notes are saved in the browser using localStorage.
